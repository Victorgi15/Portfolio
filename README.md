# Mechatronics Portfolio

Single-page React portfolio with a technical dark UI and subtle motion.

## Stack

- React + Vite
- Tailwind CSS
- Framer Motion
- lucide-react

## Setup

1. Install dependencies: `npm install`
2. Start the dev server: `npm run dev`

## Build

- Production build: `npm run build`
- Preview build: `npm run preview`

## Customize content

Edit `src/data/portfolioData.js` to update:
- Hero copy, contact details, and navigation labels
- Projects list (summary, tags, images, and GitHub links)
- Skills, method, and highlights

Add project images to `src/assets/` and reference them in the data file.

## Structure

- `src/data/portfolioData.js` - content and data objects
- `src/components/sections` - page sections
- `src/components/ui` - reusable UI primitives
- `src/components/layout` - layout elements
